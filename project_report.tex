\begin{abstract}
This report documents a complete simulation of a UAV-mounted 5G base station
over Cox's Bazar, Bangladesh, using the 3GPP TR~36.777 aerial Urban Macro channel
model. Population-weighted coverage is computed on a $40\times40$ ground grid
($\pm 5$~km) with \textbf{WorldPop constrained population count} (2026 projection,
R2025A release; 100~m, clipped from the uploaded Bangladesh raster
\texttt{bgd\_pop\_2026\_CN\_100m\_R2025A\_v1.tif}).
Optimal altitudes $h^\ast$ are found for Sub-6~GHz (3.5~GHz) and mmWave (28~GHz),
with sensitivity over transmit power and SINR threshold. All tables and figures
reflect the latest full pipeline execution.
\end{abstract}

\section{Introduction and Motivation}
%=============================================================================

\subsection{Problem statement}
A UAV base station must fly at altitude $h$ that balances:
\begin{itemize}[noitemsep]
  \item \textbf{Higher $h$}: more line-of-sight (LoS), wider footprint;
  \item \textbf{Higher $h$}: longer path $\rightarrow$ weaker edge signals;
  \item \textbf{Higher $h$}: more co-channel interference (simplified model).
\end{itemize}
Coverage versus altitude is typically non-monotonic with an interior optimum $h^\ast$.

\subsection{Regulatory context (Bangladesh)}
While current Civil Aviation Authority of Bangladesh (CAAB) rules restrict routine
beyond-visual-line-of-sight (BVLOS) UAV operations, this study establishes
\textbf{physical coverage limits} for future regulatory frameworks, tethered
aerostat deployments, or cyclone and emergency special-authorization scenarios
in coastal Bangladesh. Results should be read as a population-weighted
upper-bound simulation rather than an operational flight plan.

\subsection{Study area and population data}
\textbf{Cox's Bazar, Bangladesh} ($21.43^\circ$N, $91.98^\circ$E), coastal flat terrain.
Coverage is weighted by \textbf{where people live}:
\begin{itemize}[noitemsep]
  \item \textbf{Active dataset:} WorldPop constrained population count (2026 projection,
        R2025A; 100~m), clipped to $91.93^\circ$--$92.03^\circ$E, $21.38^\circ$--$21.48^\circ$N
        $\rightarrow$ \texttt{data/worldpop\_coxsbazar.tif}.
  \item \textbf{Parent file:} \texttt{bgd\_pop\_2026\_CN\_100m\_R2025A\_v1.tif} (full Bangladesh).
\end{itemize}

\subsection{Software stack}
\texttt{uav\_optimizer.py}, \texttt{population\_data.py}, \texttt{main.py},
\texttt{visualizations.py}, \texttt{sensitivity.py}, \texttt{live\_viz.py},
\texttt{scripts/clip\_worldpop.py}, \texttt{check\_worldpop.py}.

\subsection{Coverage and optimization}
Control coverage $C_{\mathrm{ctrl}}(h)$ and data coverage $C_{\mathrm{data}}(h)$ use
$\gamma_{\mathrm{th}}=0$~dB and $\gamma_{\mathrm{th,data}}=3$~dB respectively.
Soft coverage $C_{\mathrm{soft}}(h)$ (sigmoid) is population-weighted over the grid.
Interference follows an altitude-driven aerial model (TR~36.777 Sec.~5.2.2) with a
constant neighbor-cell margin; altitude is capped by a simplified battery/endurance model.
The optimizer maximizes $C_{\mathrm{soft}}(h)$ with \texttt{scipy.optimize.minimize\_scalar},
validated against the altitude sweep.

  \item User raster: \texttt{bgd\_pop\_2026\_CN\_100m\_R2025A\_v1.tif} (EPSG:4326, Bangladesh).
  \item \texttt{python scripts/clip\_worldpop.py} $\rightarrow$
        \texttt{data/worldpop\_coxsbazar.tif} (120$\times$120 cells, nodata cleaned).
  \item \texttt{population\_data.load\_population\_weights()} samples raster at each grid
        lon/lat; weights normalized to $[0,1]$.
  \item Verification: \texttt{python check\_worldpop.py} $\rightarrow$
        \texttt{Source: worldpop:worldpop\_coxsbazar.tif}.
\end{enumerate}

\section{Conclusions}
%=============================================================================

\begin{enumerate}
  \item With \textbf{WorldPop 2025A} weights, Sub-6 $h^\ast = 221$~m delivers
        \textbf{73.4\%} control coverage and \textbf{55.3\%} data coverage at 3.5~GHz
        ($P_{\mathrm{tx}}=27$~dBm, control $\gamma_{\mathrm{th}}=0$~dB, data $3$~dB),
        with $h_{\max}=300$~m enforced by the battery/endurance model.
  \item mmWave optimum is \textbf{120~m} ($\Delta h^\ast = -101$~m) with only
        \textbf{4.4\%} control coverage---strong band dependence for UAV placement.
  \item Altitude-driven interference (not user-distance-dependent) and endurance
        constraints produce physically consistent SINR and feasible altitude bounds.
  \item Dual-threshold sensitivity: control $\gamma_{\mathrm{th}}$ sweep (Fig.~4) and
        data $\gamma_{\mathrm{th,data}}$ sweep (Fig.~5) show $h^\ast$ falling as
        thresholds tighten; at 27~dBm / 0~dB control, $h^\ast \approx 222$~m.
\end{enumerate}
